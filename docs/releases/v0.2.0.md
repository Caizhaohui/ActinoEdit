# ActinoEdit v0.2.0

**Release Date:** 2026-07-01

## Highlights

- Complete local database support with genes table
- Full CRUD operations in Web UI for projects and saved designs
- Packaging improvements for easier distribution
- Version 0.2.0

## New Features

### Database
- Added `genes` table
- `actinoedit db import-genome` now saves gene annotations
- New commands: `list-genomes`, `list-genes`
- Project deletion support

### Web Interface
- Projects browser page now supports:
  - Create new projects
  - Delete projects
  - Save current design results to DB
  - Browse genes for imported genomes
- Quick "Save to Local DB" button in design results panel
- Auto-save of reports still available

### Packaging & Build
- Version updated to 0.2.0
- PyInstaller builds now bundle `examples/` (profiles + demo data)
- Updated build scripts and .spec file
